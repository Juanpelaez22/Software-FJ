# Software FJ Management System

## Description

Software FJ Management System is a desktop application developed in Python using the Tkinter library. The application was created as the Phase 4 project for the Object-Oriented Programming course at Universidad Nacional Abierta y a Distancia (UNAD).

The system allows users to manage customers, services and reservations while applying object-oriented programming principles, exception handling and logging techniques.

---

## Main Features

- Customer management
  - Add customers
  - Delete customers

- Service management
  - Add services
  - Delete services

- Reservation management
  - Create reservations
  - Confirm reservations
  - Cancel reservations

- Exception handling
  - Custom exceptions
  - Try / Except
  - Try / Except / Else / Finally
  - Exception chaining

- Logging system
  - Stores system events
  - Stores application errors
  - Records user operations

## How to Run

1. Clone the repository.

```bash

```

2. Open the project using Visual Studio Code.

3. Run the application.

```bash
python Software_FJ.py
```
